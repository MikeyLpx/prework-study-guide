HTML

  • HTML elements work together to a strong structure and framework for a webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • Elements that hold all the children are known as the parent element.
  • Semantic elements describe the content and role it plays on a webpage.
  • The < a > element creates links to the same webpage or other webpages.
  • The href attribute points to the URL for the link.

CSS

  • CSS adds color and style to our webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Whe using an external CSS style a link element is needed in the HTML's head.
  • Command touch style.css is used to create a CSS style sheet.
  • Declarations include the CSS property and the value we want to assign to that property
  • Sectors can be combined into a single CSS rule by using a comma.
  • The rel attribute specifies the relationship between the current document and the linked document resource
  • The href attribute specifies the location of the external source.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch.
  • git commit -m "descriptive message": commit your staged content as a new commit snapshot
  • git pull: fetch and merge any commits from then tracking remote branch
  • git push origin branch name: pushes changes from local branch to the remote branch

Javascript

  • JavaScript adds functionality to our webpage.
  • A variable is named a container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script
  • A variable is a named container that stores data in our code.
  • Console.log is used to output a message onto the web console.
  • The JavaScript file is linked with the body element.
  • When using and if statement, code will only execute if the statement meets a condition or is true.
  • A value is falsy if it has one of these values:0,-0,on,"",null,undefined,NaN, or a boolean data type with the value false
  • You can have more than one condition by using else if.
  • An array is a single variable used to hold a group of data.
  • A for loop prerforms a task on all items in an array by allowing a single block of code to be executed repeatedly.
  • A function is a set of instructions that tells the computer how to perform a certain task.
  • Functions do not automatically execute when the JavaScript file is run.
